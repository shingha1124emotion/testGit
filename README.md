## 프로젝트 개요

KT UX Design System은 디자인 토큰 기반의 멀티 플랫폼 디자인 시스템으로, Figma에서 정의된 디자인을 각 플랫폼에 맞게 자동으로 변환하고 배포합니다.

### 주요 기능

- 📱 멀티 플랫폼 지원 (Web, iOS, Android)
- 🔄 Style Dictionary 기반 토큰 변환
- 📦 플랫폼별 패키지 관리 (pnpm workspace)
- 🚀 자동화된 빌드 및 배포 파이프라인
